Introduction

This page will be a very loose and informal introduction to the Java programming language. It will serve as a good stepping stone into coding in general. Expect the guide to be informative in nature but not extremely technical.

The reason I chose Java (over Python or any other language) (for the workshop) is due to the "strictness" of the language. It teaches good habits whilst also being a good first language. However, the workshop is intended to be language agnostic - meaning that the core concepts learnt will be applicable across any language of your choice!

Prerequisites

Getting started is quite simple actually. First, you need to install Java SE from here. Then, to write/run code, you just need to install Eclipse for Java Developers (it can be found here). Its a simple software which allows you to write your code and run it, all in one place.

An extra, supplementary note: running Java code without Eclipse can be a bit hard, since you have to manually run it using multiple commands.

Getting Started (on Eclipse)

Just like we need to open a notebook to write something, we need to open a 'project' and then create a 'class' on Eclipse to write our code.

Thus, we click on 'Create a Java Project', naming it something of our choice. Afterwards, we right click on the project we just made and create a class. When making a class, we have to make sure we checkmark the first box which gives us the "main method".

Hello World

Probably the first thing anyone does when learning any language is write a program that says "Hello World!."

Its actually a good starting point simply because you are actually 'creating' something! Almost like a robot saying its first sentence, greeting the world!

To make Eclipse say "Hello World!" we actually need just 1 statement! After creating a project and a class, copy and paste this into the body of the class and run it. Then look at the console!

System.out.println("Hello World!");

Variables

A program is not as smart as you might think (for better or for worse!). You need to specify what information you are feeding into the program.

The data you can write in the program is known as 'variables'. Just like in math, we have 'x' and 'y' as variables, we have variables in programming.

Variables can be any combination of letters and numbers, except a few rules that exist to reduce confusion. Some examples are below:

helloWorld
abc
x
y1

These variables have types as well! You specify those for the program to understand. A list of variables which you will be using is given below:

  • int - Integers
  • double - Decimal Numbers
  • boolean - true and false
  • String - Combination of letters, words, numbers - practically any written sentence!
  • char - Single letters

if-else Statements

It might be a very new concept to you, but it is as simple as it sounds!

Imagine I told you to do something, but there is a condition to it. What I mean is, what if I told you to carry out this instruction: "If today is Saturday, do not go to school." The computer can also understand this, if you can tell it in a certain way! For example:

if (today == "Saturday") {

goToSchool = false;

}

Now lets add the other case - what if it isn't Saturday? We can write this as: "but if it is not Saturday, do go to school." We just add the code below to the previous one.

else {

goToSchool = true;

}
